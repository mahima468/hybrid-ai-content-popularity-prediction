# hybrid-ai-content-popularity-prediction
Hybrid AI System for Fake Engagement Detection & Sentiment-Driven Content Popularity Forecasting

This project analyzes digital content engagement and predicts future popularity using machine learning.

## Features
- Fake engagement detection
- Sentiment analysis on user comments
- Content popularity prediction
- Virality scoring
- Interactive dashboard

## Tech Stack
Python, FastAPI, Scikit-learn, NLP, React, Streamlit

## Modules
1. Sentiment Analysis
2. Fake Engagement Detection
3. Popularity Prediction
4. Dashboard Visualization
